Data usage:

That data is included in an xlsx file, which is compatible with Excel 2016. The file includes both the raw data as well as the plots. 

All data was dowloaded from free to access websites. The data can be freely copied and used without restriction. 


Data sources: 

Data on number of blocks per day retrieved from bitcoinity.org. The link  https://data.bitcoinity.org/bitcoin/block_time/all?t=l gives access to a direct download. 

Remaining data retrieved from blockchain.info. To obtain the data, request an API key by creating an account through this link: https://www.blockchain.com/api. As of 2017, no payment was required for dowloading all the Bitcoin blockchain data at a daily resolution.
