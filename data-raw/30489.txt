This folder contains the code to replicate the figures in the paper "Dollar Safety and the Global Financial Cycle" by Zhengyang Jiang, Arvind Krishnamurthy, and Hanno Lustig. The code is written in Matlab, with the exception of an auxiliary code to load data, which is written in R.

Figure 1: import_basis.R + plot_basis.m
Figure 2-4, 6-8: irf_generate_plots.m
Figure 5: plot_wealth.m
Figure 9: irf_generate_plots_3countries.m
